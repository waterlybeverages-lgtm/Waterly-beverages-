
Waterly - Static Website (15 pages)
Files in this folder:
- index.html (home)
- products.html, product-1.html, product-2.html, product-3.html
- about.html, impact.html, shop.html, contact.html
- faq.html, legal.html, press.html, blog.html, investor.html, terms.html
- /assets: placeholder SVG images and mockup SVGs

How to use:
Open index.html in a browser. Replace assets/*.svg with real photos and mockups.
This template uses Tailwind CDN for quick styling. For production, replace CDN with compiled CSS and optimize images.

Design notes:
- All text is in Hindi/Hinglish and ready to edit.
- Buttons link to shop.html; replace with payment integration (Stripe/PayU) and backend for subscriptions.
